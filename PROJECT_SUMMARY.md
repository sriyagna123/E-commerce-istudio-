## 🔧 Easy Customization

### Change Colors
Edit CSS variable in `css/style.css`:
```css
:root {
  --primary-color: #007bff;  /* Change this */
  --secondary-color: #6c757d;
  /* ... etc ... */
}
```

### Add Products
Edit `data/products.json`:
```json
{
  "id": 13,
  "name": "Your Product",
  "category": "Electronics",
  "price": 99.99,
  "originalPrice": 149.99,
  "rating": 4.5,
  "image": "url",
  "description": "Your description"
}
```

### Modify Text/Copy
Update text in HTML pages directly. All content is easy to find and modify.
